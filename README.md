# nivelacion-crear-un-sitio-web-ayala-lautaro
Proyecto de Nivelación - Sitio Web Multi-Entorno
Este proyecto consiste en una aplicación web simple que permite interactuar con una caja de texto y cambiar dinámicamente el esquema de colores del sitio. Se presenta en tres implementaciones distintas para demostrar el manejo de diferentes entornos de ejecución.

Cómo ejecutar cada implementación
1) Implementación con JavaScript Puro (Sin servidor)
Esta versión no requiere ninguna instalación previa ni servidor activo.

Ubicación: /implementacion1

Instrucciones: 1. Localiza el archivo index.html.
2. Hacé doble clic sobre él o arrástralo a tu navegador.
3. Listo ya funciona!

2) Implementación con Node.js (Servidor Express)
Esta versión requiere tener instalado Node.js.

Ubicación: /implementacion2

Instrucciones:

Abrí una terminal en la carpeta /implementacion2.

Instala las dependencias necesarias:


npm install
Inicia el servidor:

node servidor.js
Abre tu navegador en: http://localhost:3000

3) Implementación con Python servidor flask
Esta versión requiere tener instalado Python.

Ubicación: /implementacion3

Instrucciones:

Abre una terminal en la carpeta /implementacion3.

Instala el framework Flask:


pip install flask
(Si el comando pip falla, intenta con python -m pip install flask)

Inicia el servidor:


python app.py
Abre tu navegador en: http://localhost:5000
